## 一、什么是 Google Gemini？

Google Gemini 是 Google DeepMind 推出的原生多模态大模型家族，能统一理解与生成文本、图像、音频、视频与代码。当前主力版本包含：

- Gemini Ultra（Gemini Advanced 的核心）
- Gemini Pro（迭代到 1.5 与 2.5 Pro）
- Gemini 3 Pro Preview（新一代预览版）
- Gemini Nano（端侧轻量版）
Gemini 以超长上下文（最高可达百万级 tokens）与 Google 生态整合见长，适合长文档处理与多模态复杂任务。

## 七、常见问题（FAQ）

- 要“翻墙”吗？中文平台通常不需要；官网入口可能需要合适网络。
- Gemini 免费吗？标准版免费，高阶能力需订阅；第三方常有限额或增值功能。
- 数据安全吗？第三方与官方政策独立，勿输入敏感信息。
- 上下文窗口是什么？一次可处理的信息上限。Gemini 1.5/2.5 支持百万级实验窗口，适合长文档与长视频。
- 开发者如何用？通过 Google AI Studio 或 Google Cloud Vertex AI 获取密钥与调用。
